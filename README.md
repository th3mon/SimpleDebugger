# SimpleDebugger

SimpleDebugger works like console.log(). It puts messages into DOM.

## Getting Started

### Install dependencies
```sh
npm i
```

### Build

#### dev
```sh
npm run build:dev
```

#### dist
```sh
npm run build:dist
```

### Develop
```sh
npm start
```
