# SimpleDebugger

SimpleDebugger works like console.log(). It puts messages into DOM.

## Getting Started

### Install dependencies
```sh
npm i
```

### Build
```sh
npm run build
```

### Develop
```sh
npm run serve
```

## Libraries and Tools
